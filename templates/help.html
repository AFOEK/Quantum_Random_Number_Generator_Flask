  • Iteration -> How many digit you want
  • Shots -> This will increase your accuracy number generated.WARNING IT WILL SLOW THE SERVER IF YOU APPLY >= 4096 PROPOTIONAL WITH HOW MANY NUMBER DIGIT YOU GENERATE
  • Qubit Count -> How many qubit you want it will generate how many digit you have, in conjunction with iteration
  • Auto Iteration -> How many number you want generate
  • Result -> It will print out result (number) to text area below
  • Binary Form -> It will generate binary form from representative number
    [0110010]
  • Digit -> How many digit from generated number
  • All info -> Print out all information above
  • Bar -> Display bar chart for Auto Generate Statistics
  • Heatmap -> Display heatmap chart for Auto Generate Statistics
  • Scatter map -> Display scatter chart for Auto Generate Statistics
  • Generate -> It will generate number according with Iteration, Shots, Qubit Count, and output you selected or set
  • Clear -> Clear all output and set all option to defualt value
  • Auto Generate - > It will generate how many number you give
  • Auto Generate Statistics -> it will show chart according what you choose.
